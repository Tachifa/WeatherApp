
Простое приложение для прогноза погоды с рекомендациями на Python.

 О приложении
Приложение показывает текущую погоду и дает рекомендации что надеть или взять с собой.

 Функции
- Текущая температура
- Скорость ветра
- Влажность
- Уровень магнитных бурь
- Рекомендации (зонт, куртка, таблетка от головы)
